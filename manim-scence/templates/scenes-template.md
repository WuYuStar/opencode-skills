### 卡片创建标准代码
```python
# 创建卡片（必须验证不重叠）
card = Rectangle(width=[W], height=[H])
card.move_to([X, Y, 0])
card.set_stroke([COLOR], width=2, opacity=0.3)

# 在卡片内放置内容
element = [创建元素]
element.move_to(card.get_center())  # 或其他相对位置
```
