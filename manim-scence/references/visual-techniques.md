# 数学动画视觉技法
用于讲解数学概念的高效可视化范式

## 核心原则

### 1. 渐进式呈现
绝不一次性展示全部内容，循序渐进地增加复杂度。
**同时避免文字堆叠**：同一屏幕内文字不超过3行，公式不超过1组核心表达式，多余说明用旁白承载，不堆在画面上。

**反面示例**：直接显示完整方程
**正面示例**：逐项构建方程，同时解释每一部分

```
场景流程：
1. 展示简单形式：f(x) = x²
2. 增加复杂度：f(x) = ax²
3. 完整形式：f(x) = ax² + bx + c
```

### 2. 变形而非替换
尽可能让物体平滑演变为新形式，而非淡出再淡入。

**反面示例**：FadeOut(equation1), FadeIn(equation2)
**正面示例**：TransformMatchingTex(equation1, equation2)

这样可以保持视觉连贯性，清晰体现不同形式之间的关联。

### 3. 色彩即含义
全程使用统一的色彩体系来传递信息。

**常用范式**：
- 输入/已知量：蓝色
- 输出/结果：绿色
- 讨论的关键项：黄色高亮
- 错误/负值：红色
- 中性/辅助内容：白色/灰色

### 4. 空间关系表意
位置本身就代表逻辑关系：
- 从左到右：变换、时间、因果
- 从上到下：层级、推导
- 中心：关注焦点
- 边缘：背景、参考信息

### 5. 简洁克制，避免文字堆叠
- 画面只放**核心公式与关键词**，解释性文字尽量用旁白
- 同一区域内文字行数≤3，公式数量≤1
- 长推导分步出现，不一次性铺满屏幕
- 重要文字留出足够间距，不挤在同一狭小区域

## 动画技法

### 强调与聚焦

**突出提示**——短暂闪烁以吸引注意力
```python
self.play(Indicate(term))
```

**圈注强调**——在重要元素周围画圈
```python
self.play(Circumscribe(equation, color=YELLOW))
```

**环绕闪烁**——关键结论揭晓时的醒目强调
```python
self.play(FlashAround(result))
```

### 方程处理

**分离项**——对特定部分上色或移动
```python
equation.set_color_by_tex("x", BLUE)
```

**分步推导**——展示每一步代数变形
```python
step1 = MathTex(r"2x + 4 = 10")
step2 = MathTex(r"2x = 6")
step3 = MathTex(r"x = 3")
# 对齐并在步骤间平滑变换
```

**代入替换**——展示数值代入变量的过程
```python
# 动画演示数字移动到变量位置
```

### 几何直观

**坐标系**：始终标注坐标轴
```python
axes = Axes(x_range=[-3, 3], y_range=[-2, 2])
labels = axes.get_axis_labels(x_label="x", y_label="f(x)")
```

**轨迹追踪**：展示点的运动路径
```python
trace = TracedPath(dot.get_center, stroke_color=YELLOW)
```

**面积可视化**：用于积分、求和
```python
area = axes.get_area(graph, x_range=[a, b], color=BLUE, opacity=0.5)
```

### 三维技法

**相机环绕**：展现三维结构
```python

```

**投影**：展示三维物体的二维投影

## 常用视觉隐喻

### 向量 → 箭头
- 位置向量：从原点出发的箭头
- 加法：首尾相接
- 数乘：拉伸/压缩

### 函数 → 机器
- 输入从一侧进入
- 内部发生变换
- 输出从另一侧出来

### 矩阵 → 线性变换
- 展示网格被整体变换
- 追踪基向量的落点
- 用行列式表示面积缩放比例

### 导数 → 斜率
- 切线与曲线相切
- 放大画面以体现局部线性
- 动画演示斜率随点移动而变化

### 积分 → 累积
- 用矩形表示黎曼和
- 动画演示宽度趋于 0
- 曲线下方的面积逐渐填充

## 场景构图

### 黄金布局

- 留白：屏幕边缘留 10-15% 安全区
- 文字与公式只放在固定区域，**不跨区堆叠、不铺满屏幕**
```
┌─────────────────────────────────────────────┐
│                安全边距 10%                 │
│  ┌─────────────────────────────────────┐  │
│  │              标题 / 上下文           │  │
│  ├─────────────────────────────────────┤  │
│  │                                     │  │
│  │                                     │  │
│  │          核心可视化主区域            │  │
│  │           （画面最大占比）           │  │
│  │                                     │  │
│  │                                     │  │
│  ├─────────────────────────────────────┤  │
│  │           方程 / 公式 / 结论         │  │
│  │          （底部 1/3 区域）           │  │
│  └─────────────────────────────────────┘  │
│                安全边距 10%                 │
└─────────────────────────────────────────────┘
```

## 时长参考

| 动作 | 常用时长 |
|------|----------|
| 简单图形生成 | 0.5–1 秒 |
| 文字/方程书写 | 1–2 秒 |
| 变换动画 | 1–2 秒 |
| 镜头移动 | 2–3 秒 |
| 停顿消化 | 0.5–1 秒 |
| 复杂动画 | 2–4 秒 |

### 节奏范式
快–快–慢–快–快–慢
