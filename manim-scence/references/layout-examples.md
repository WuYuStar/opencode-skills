# Manim Composer - 卡片布局示例

## 布局模式总览

### 1. 左右分屏（2卡片）
**适用场景**：可视化+说明

```markdown
### 卡片布局
**布局模式**：左右分屏

**卡片定义：**
| 卡片名 | 宽 | 高 | 中心 | 用途 | 颜色 |
|--------|----|----|------|------|------|
| GraphCard | 6.5 | 7.0 | (-3.5, 0) | 函数图像 | BLUE |
| TextCard | 6.5 | 7.0 | (+3.5, 0) | 说明文字 | GREEN |

**间距验证：**
- GraphCard右边界：-3.5 + 3.25 = -0.25
- TextCard左边界：+3.5 - 3.25 = +0.25
- 水平间距：0.25 - (-0.25) = 0.5单位 ✓（不重叠）
- GraphCard上边界：0 + 3.5 = 3.5
- GraphCard下边界：0 - 3.5 = -3.5
- 在垂直范围内：±4 ✓
```

**Manim代码示例：**
```python
# 创建卡片背景（带调试边框）
graph_card = Rectangle(
    width=6.5, height=7.0,
    stroke_color=BLUE, stroke_opacity=0.3,
    fill_color=BLACK, fill_opacity=0.8
).move_to((-3.5, 0, 0))

text_card = Rectangle(
    width=6.5, height=7.0,
    stroke_color=GREEN, stroke_opacity=0.3,
    fill_color=BLACK, fill_opacity=0.8
).move_to((3.5, 0, 0))
```

### 2. 网格布局（4卡片）
**适用场景**：多角度对比

```markdown
### 卡片布局
**布局模式**：2×2网格

**卡片定义：**
| 卡片名 | 宽 | 高 | 中心 | 用途 | 颜色 |
|--------|----|----|------|------|------|
| Card1 | 6.5 | 3.5 | (-3.5, +1.8) | 视角1 | BLUE |
| Card2 | 6.5 | 3.5 | (+3.5, +1.8) | 视角2 | GREEN |
| Card3 | 6.5 | 3.5 | (-3.5, -1.8) | 视角3 | ORANGE |
| Card4 | 6.5 | 3.5 | (+3.5, -1.8) | 视角4 | PURPLE |

**间距验证：**
- 水平间距：同左右分屏，0.5单位 ✓
- 垂直间距：
  - Card1下边界：1.8 - 1.75 = 0.05
  - Card3上边界：-1.8 + 1.75 = -0.05
  - 垂直间距：0.05 - (-0.05) = 0.1单位 ❌
  
**修正后：**
| Card1 | 6.5 | 3.3 | (-3.5, +1.85) | ... | ... |
| Card3 | 6.5 | 3.3 | (-3.5, -1.85) | ... | ... |

- 新垂直间距：
  - Card1下边界：1.85 - 1.65 = 0.2
  - Card3上边界：-1.85 + 1.65 = -0.2
  - 垂直间距：0.2 - (-0.2) = 0.4单位 ✓
```

### 3. 列表布局（3卡片纵向）
**适用场景**：步骤展示

```markdown
### 卡片布局
**布局模式**：列表布局（3卡片纵向）

**卡片定义：**
| 卡片名 | 宽 | 高 | 中心 | 用途 | 颜色 |
|--------|----|----|------|------|------|
| Step1 | 13.0 | 2.3 | (0, +2.6) | 步骤1 | BLUE |
| Step2 | 13.0 | 2.3 | (0, 0) | 步骤2 | GREEN |
| Step3 | 13.0 | 2.3 | (0, -2.6) | 步骤3 | ORANGE |

**间距验证：**
- Step1下边界：2.6 - 1.15 = 1.45
- Step2上边界：0 + 1.15 = 1.15
- 间距1：1.45 - 1.15 = 0.3单位 ✓

- Step2下边界：0 - 1.15 = -1.15
- Step3上边界：-2.6 + 1.15 = -1.45
- 间距2：-1.15 - (-1.45) = 0.3单位 ✓

- 总高度：2.3×3 + 0.3×2 = 6.9 + 0.6 = 7.5 ≈ 8 ✓
```

- 左右留白：±7 ∓ 6.75 = ±0.25
- 上下留白：±4 ∓ 3.75 = ±0.25
- 四周均匀留白 ✓

## 布局设计检查清单

**设计阶段：**
- [ ] 确定场景主要内容类型（图形/文字/对比）
- [ ] 选择合适的布局模式
- [ ] 确定卡片数量和尺寸
- [ ] 计算并验证所有间距 ≥ 0.2

**验证阶段：**
- [ ] 水平方向：任意两卡片左右边界差 ≥ 0.2
- [ ] 垂直方向：任意两卡片上下边界差 ≥ 0.2
- [ ] 画布覆盖：总宽度 ≈ 14，总高度 ≈ 8
- [ ] 中心对称：复杂布局检查视觉平衡

**实现阶段：**
- [ ] 添加调试边框（stroke_opacity=0.3）
- [ ] 文字大小控制在24-36pt
- [ ] 图形不超出卡片边界
