# 分镜示例
3Blue1Brown 风格视频的分镜拆解示例

## 示例1：讲解点积

### 分镜1：提出问题
**时长**：约30秒
**目的**：用悬念吸引观众

**视觉元素**
- 绘制两个向量 a 和 b（箭头形式）
- 点积公式：$\boldsymbol{a} \cdot \boldsymbol{b} = |\boldsymbol{a}||\boldsymbol{b}|\cos\theta$
- 问号动画

**内容**
画面以两个向量开场，展示公式，抛出问题：
“为什么把分量相乘再相加，会和向量之间的夹角有关？”

**旁白提示**
语气：好奇、略带疑惑。强调这个公式看起来很不自然、很随意。

**技术要点**
- 用 Arrow 绘制向量
- 用 MathTex 书写公式
- 对 $\cos\theta$ 项使用 Indicate() 强调

---

### 分镜2：几何意义
**时长**：约90秒
**目的**：展示投影解释

**视觉元素**
- 水平蓝色向量 a
- 倾斜绿色向量 b
- b 在 a 上的投影（虚线）
- 直角标记
- 长度标注

**内容**
展示点积等于 $|\boldsymbol{a}|$ 乘以 b 在 a 上的投影长度。
动画演示投影垂直落下，从几何上说明其等于 $|\boldsymbol{a}||\boldsymbol{b}|\cos\theta$。

**旁白提示**
“点积衡量的是一个向量在另一个向量方向上的贡献程度。”

**技术要点**
- 投影用 DashedLine
- 使用 RightAngle 物体绘制直角
- 用 animate.rotate() 展示不同夹角

---

### 分镜3：数值关联
**时长**：约60秒
**目的**：建立几何与代数的联系

**视觉元素**
- 坐标网格
- 向量 $\boldsymbol{a} = [a_1,a_2]$
- 向量 $\boldsymbol{b} = [b_1,b_2]$
- 高亮分量

**内容**
在网格上展示带分量的向量，说明为什么 $a_1b_1+a_2b_2$ 与几何解释等价，并用具体数值演示。

**旁白提示**
缓慢推导计算过程。
“我们来看看为什么代数形式和几何形式是一致的。”

**技术要点**
- 使用 NumberPlane 或 Axes
- 用 Brace 标注分量
- 用 TransformMatchingTex 展示公式推导步骤

## 示例2：傅里叶级数入门

### 分镜1：吸引点
**时长**：约45秒
**目的**：展示出人意料的结果

**视觉元素**
- 方波（棱角分明）
- 光滑正弦波的叠加
- 二者之间的变形动画

**内容**
“你竟然可以用完全光滑的正弦波，拼出有尖锐拐角的方波。”
先展示结果，再承诺讲解原理。

**旁白提示**
语气：惊叹、略带不可思议。让观众感到反直觉、很神奇。

**技术要点**
- 用 ParametricFunction 绘制波形
- 用 Transform 动画实现形态过渡
- 可展示1、3、5项逐步叠加的过程

---

### 分镜2：基本单元
**时长**：约120秒
**目的**：介绍正弦波作为基函数

**视觉元素**
- 单个正弦波
- 频率可视化（振动更快）
- 振幅可视化（更高/更矮）
- 相位可视化（左右平移）

**内容**
介绍三个参数：频率、振幅、相位。分别单独展示，再组合演示。

**旁白提示**
节奏放慢。
“正弦波有三个可以调节的旋钮……”

**技术要点**
- 用 ValueTracker 制作参数动画
- 用 Updater 让波形随参数实时变化
- 为每个参数添加标签

### 分镜3：叠加原理
**时长**：约90秒
**目的**：展示波可以相加

**视觉元素**
- 两个不同颜色的正弦波
- 它们的和（第三种颜色）
- 逐点相加的可视化

**内容**
展示波的叠加就是在每个位置上把高度相加。用两个特定频率的波演示合成效果。

**旁白提示**
“波的叠加很简单——在每个点上，直接把高度加起来就行。”

**技术要点**
- 用 VGroup 管理三条函数曲线
- 用竖线在特定 x 位置展示加法过程
- 动画演示逐点相加

## 示例3：矩阵与线性变换

### 分镜1：网格变换
**时长**：约60秒
**目的**：建立直观视觉基础

**视觉元素**
- 二维坐标网格（NumberPlane）
- 基向量 $\hat{\boldsymbol{i}}$ 和 $\hat{\boldsymbol{j}}$（彩色箭头）
- 网格线整体变换

**内容**
展示一个网格，高亮 $\hat{\boldsymbol{i}}=(1,0)$ 和 $\hat{\boldsymbol{j}}=(0,1)$。
施加一个变换，观察整个网格移动，并追踪基向量的落点。

**旁白提示**
“观察施加变换后网格的变化。注意每个点是如何移动的。”

**技术要点**
- 使用带清晰网格线的 NumberPlane
- 使用 apply_matrix() 方法
- 基向量保持视觉上明显区分

---

### 分镜2：基向量决定一切
**时长**：约90秒
**目的**：点明核心洞见

**视觉元素**
- 变换后的 $\hat{\boldsymbol{i}}$ 和 $\hat{\boldsymbol{j}}$
- 任意向量 v 作为二者的组合
- $\boldsymbol{v}=x\boldsymbol{i}+y\boldsymbol{j}$ 的可视化

**内容**
展示只要知道 $\hat{\boldsymbol{i}}$ 和 $\hat{\boldsymbol{j}}$ 变换后的位置，就能确定**任意**向量的位置。
因为 $\boldsymbol{v}=x\boldsymbol{i}+y\boldsymbol{j}$，所以变换后 $\boldsymbol{v}'=x\boldsymbol{i}'+y\boldsymbol{j}'$。

**旁白提示**
“这就是核心结论……”
在揭示前制造一点期待感。

**技术要点**
- 向量加法动画（首尾相接）
- 系数缩放动画
- 用 TransformMatchingShapes 展示组合关系

# 转场模式

### 聚焦缩放
```
完整画面 → 放大细节 → 讲解 → 拉回全景
```

### 并列构建
```
左侧空白 | 右侧空白
向左侧添加内容 | 形成对比
向右侧添加内容 | 建立关联
```

### 连续变换
```
物体A → 变形 → 物体B → 变形 → 物体C
（全程保持视觉连贯）
```

### 重置重构
```
复杂画面 → 清除/淡出大部分内容 → 聚焦单个元素 → 重新构建新复杂度
```
