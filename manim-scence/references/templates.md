# Manim Composer - 模板参考

## 过渡设计

### 模板结构

```markdown
### 过渡设计

**场景N → 场景N+1：**
- 过渡方式：[Transform/Fade/CameraPan/Zoom]
- 延续元素：[哪些元素保持并变化]
- 新元素：[哪些新元素入场]
- 时长：X秒
- 视觉线索：[观众应该注意到的关联]
- 情感节奏：[紧张/舒缓/惊讶/期待]
```

---

### 示例1：内容形变（Transform）

```markdown
**场景1 → 场景2：**
- 过渡方式：Transform（形变过渡）
- 延续元素：方波图形保持形状，但颜色从灰色变为蓝色
- 新元素：逐渐叠加第一个正弦波分量
- 时长：1.5秒
- 视觉线索：方波"拆解"成正弦波的瞬间
- 情感节奏：从困惑（这是什么？）到好奇（它在变化！）
```

**Manim代码思路：**
```python
# 延续元素变换
self.play(
    square_wave.animate.set_color(BLUE),
    run_time=0.5
)

# 新元素入场
self.play(
    Create(first_sine_wave),
    run_time=1.0
)

# 展示叠加效果
self.play(
    Transform(square_wave, partial_sum),
    run_time=1.5
)
```

### 示例2：镜头平移（Camera Pan）

```markdown
**场景2 → 场景3：**
- 过渡方式：Camera Pan（镜头向右平移）
- 延续元素：时域波形从左侧移到右侧
- 新元素：左侧出现对应的频谱图
- 时长：1.2秒
- 视觉线索：时域和频域的对应关系
- 情感节奏：发现新视角的惊喜
```

---

### 示例3：放大深入（Zoom）

```markdown
**场景3 → 场景4：**
- 过渡方式：Zoom In（放大到局部）
- 延续元素：函数图像的特定区域
- 新元素：放大后显示切线和斜率计算
- 时长：1.0秒
- 视觉线索：从整体到局部的深入
- 情感节奏：专注、深入理解的紧张感
```

---

### 示例4：淡入淡出（Fade）

```markdown
**场景5 → 场景6：**
- 过渡方式：Fade Out/Fade In（完全切换）
- 延续元素：无（场景主题完全转变）
- 新元素：全新的视觉元素
- 时长：1.0秒淡出 + 0.5秒停顿 + 1.0秒淡入
- 视觉线索：明显的场景切换，让观众知道进入了新章节
- 情感节奏：停顿、呼吸、准备接受新内容
```

## 信息整理模板

### 主题知识整理

```markdown
主题：[概念名称]

核心知识点：
1. [知识点1] - 重要性：★★★★★ - 可视化难度：易/中/难
2. [知识点2] - 重要性：★★★★☆ - 可视化难度：易/中/难
3. ...

可视化思路：
- [思路1]：具体如何画图/动画展示
- [思路2]：最佳展示角度/视角
- [思路3]：可能的动画序列

常见误区：
- [误区1]：学生容易误解的地方 + 如何澄清
- [误区2]：...

应用实例：
- [应用1]：工程/物理中的实际应用
- [应用2]：日常生活中的例子
- [应用3]：与其他学科的联系

"顿悟时刻"候选：
- [时刻1]：最容易让学生恍然大悟的切入点
- [时刻2]：...
```

## 场景内容描述模板

### 内容描述结构

```markdown
### 内容描述

**开场**（0-20%时长）：
- 画面状态：[初始画面描述]
- 旁白开场：[吸引注意力的第一句话]
- 目标：建立[情感状态]

**发展**（20-70%时长）：
- 画面变化：[逐步展示的内容]
- 旁白展开：[解释、引导、提问]
- 关键帧：[必须出现的特定画面]

**高潮**（70-85%时长）：
- 画面焦点：[最精彩的视觉瞬间]
- 旁白高潮：[核心洞见的揭示]
- "顿悟时刻"：[具体哪一秒/哪个画面]

**结尾**（85-100%时长）：
- 画面收尾：[如何收场]
- 旁白过渡：[为下一个场景铺垫]
- 情感释放：[满足/期待/思考]
```

---

### 示例：傅里叶变换场景3

```markdown
### 内容描述

**开场**（0-15秒）：
- 画面状态：左侧卡片显示灰色方波，右侧卡片显示第一个正弦波
- 旁白开场："我们已经知道，方波可以被分解成一系列正弦波。但具体是怎么做到的呢？"
- 目标：建立好奇

**发展**（15-80秒）：
- 画面变化：逐步叠加3次、5次、7次谐波，方波越来越清晰
- 旁白展开："首先，我们有一个基础的正弦波。然后加上频率是它的3倍的正弦波..."
- 关键帧：叠加到第7次谐波时，方波形状基本清晰

**高潮**（80-105秒）：
- 画面焦点：快速展示叠加到第15次谐波，方波几乎完美
- 旁白高潮："看！随着我们加入越来越多的正弦波，叠加的结果越来越接近原来的方波！"
- "顿悟时刻"：第90秒，当观众意识到"无限多个正弦波可以完美重构方波"

**结尾**（105-120秒）：
- 画面收尾：定格在叠加状态，显示频谱图在右侧出现
- 旁白过渡："这就是傅里叶级数的魔力。但如果我们想知道每个频率的贡献有多大呢？"
- 情感释放：惊讶转为期待
```
