# Manim Composer - 示例与模板

## 询问示例

### 示例1：傅里叶变换

**用户：** 帮我做一个关于傅里叶变换的视频。

**你：** 好的！为了做出最适合你需求的视频，我需要了解几个细节：

1. **你的目标观众是什么水平？**
   - 高中生（基础数学）
   - 大学工程类学生（学过微积分）
   - 信号处理专业人士

2. **你希望视频多长？**
   - 5-8分钟：只讲核心概念
   - 10-15分钟：完整讲解+直观理解
   - 20-30分钟：深度探索+应用

3. **你希望风格偏重：**
   - 几何直觉（图形动画为主）
   - 还是代数推导（公式推导为主）？

4. **你希望包含实际应用吗？**（比如音频处理、图像压缩等）

### 示例2：微积分基础

**用户：** 我想做个讲解导数的视频。

**你：** 好的！我来帮你设计。先确认几个关键点：

1. **观众背景**：
   - 完全没学过微积分的新手？
   - 还是学过一些但需要巩固？

2. **你心目中的"顿悟时刻"是什么？**
   - 比如"导数就是瞬时变化率"的直观理解？
   - 或者"导数的几何意义是切线斜率"？

3. **需要避开的内容**：
   - 极限的严格定义？
   - 高阶导数？

## 知识整理

### 示例：傅里叶变换

```markdown
主题：傅里叶变换

核心知识点：
1. 任何周期函数都可以表示为正弦波的叠加 - 重要性：★★★★★
2. 傅里叶级数（离散情况）- 重要性：★★★★★
3. 傅里叶积分（连续情况）- 重要性：★★★★☆
4. 频域 vs 时域的对应关系 - 重要性：★★★★★
5. 正交性原理 - 重要性：★★★☆☆

可视化思路：
- 方波分解：用动画展示方波如何由多个正弦波叠加而成
- 旋转矢量：用复平面上的旋转矢量表示正弦波
- 频谱图：展示不同频率成分的强度
- 时频对应：同一信号在时域和频域的对照展示

常见误区：
- 误解1：认为傅里叶变换"改变"了信号，而不是"展示"了信号的不同角度
- 误解2：混淆周期信号的傅里叶级数和非周期信号的傅里叶变换
- 误解3：不理解频谱中负频率的物理意义

应用实例：
- 音频处理：MP3压缩利用人耳对不同频率的敏感度
- 图像压缩：JPEG利用傅里叶变换去除高频细节
- 信号处理：从噪声中提取有用信号
- 物理：分析振动模式、量子力学中的波函数
```

## 知识分配

### 示例：傅里叶变换（6场景版本）

```markdown
场景1：问题引入（60秒）
- 知识点：信号处理的实际困惑
- 可视化：真实世界的复杂信号（音频波形、图像）
- 目标：建立"如何分析复杂信号"的困惑
- 钩子："你能听出这段音乐里的所有乐器吗？"

场景2：背景铺垫（75秒）
- 知识点：周期性函数、正弦波基础
- 可视化：单位圆上的旋转矢量、正弦波生成
- 目标：统一知识基础，确保观众理解正弦波

场景3：核心概念（上）（120秒）
- 知识点：傅里叶级数（离散情况）
- 可视化：方波逐步分解为正弦波（1次谐波→3次→5次→...）
- 目标：初步理解"任何周期函数都是正弦波叠加"
- 顿悟时刻：当叠加的正弦波越来越接近方波时

场景4：核心概念（下）（120秒）
- 知识点：频域表示、频谱图
- 可视化：从时域波形平滑过渡到频域柱状图
- 目标：理解频域是另一种观察角度

场景5：直观理解（90秒）
- 知识点：不同频率正弦波的正交性（直观理解，非数学证明）
- 可视化：展示为什么不同频率的正弦波"不干扰"彼此
- 目标：让观众"看见"为什么分解是可能的

场景6：总结升华（75秒）
- 知识点：应用场景回顾（MP3、JPEG、信号处理）
- 可视化：快速展示各种应用场景的图标
- 目标：从困惑到满足，启发进一步探索
```

### 示例：导数概念（8场景扩展版）

```markdown
场景1：问题引入（45秒）
- 知识点：瞬时速度的现实需求
- 可视化：汽车速度表、运动轨迹
- 钩子："如何计算某一瞬间的速度？"

场景2：背景铺垫（60秒）
- 知识点：平均速度 vs 瞬时速度
- 可视化：时间间隔越来越小的动画

场景3：核心概念（上）（90秒）
- 知识点：导数的极限定义
- 可视化：割线逐步逼近切线

场景4：核心概念（中）（90秒）
- 知识点：导数的几何意义
- 可视化：各种函数在特定点的切线

场景5：核心概念（下）（90秒）
- 知识点：可导与连续的关系
- 可视化：尖点、间断点的例子

场景6：常见误区（60秒）
- 知识点：导数不是"微小变化"，而是变化率
- 可视化：对比错误理解和正确理解

场景7：实际应用（90秒）
- 知识点：最优化问题、相关变化率
- 可视化：实际应用案例（盒子最大容积等）

场景8：总结升华（60秒）
- 知识点：导数是函数在某点的"局部线性近似"
- 可视化：切线近似函数图像
```
