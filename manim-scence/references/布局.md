# 场景布局设计规范

## 布局设计流程

### 第1步：分析场景内容
确定场景需要展示什么：
- 主要概念/公式
- 可视化图形
- 说明文字
- 对比内容

### 第2步：选择布局模式

#### 模式A：左右分屏（2卡片）
**适用场景**：可视化+说明
```
┌─────────────────────────────────────┐
│  ┌───────────────┐ ┌─────────────┐  │
│  │               │ │             │  │
│  │   左卡片      │ │   右卡片    │  │
│  │   宽6.5       │ │   宽6.5     │  │
│  │   (可视化)    │ │   (文字)    │  │
│  │               │ │             │  │
│  └───────────────┘ └─────────────┘  │
└─────────────────────────────────────┘
```

**坐标计算**：
- 左卡片：中心(-3.5, 0)，宽6.5，右边界在-0.25
- 右卡片：中心(+3.5, 0)，宽6.5，左边界在+0.25
- 间距：0.5单位（安全不重叠）

#### 模式B：网格布局（4卡片）
**适用场景**：多角度展示、对比
```
┌─────────────────────────────────────┐
│  ┌─────────┐      ┌─────────┐       │
│  │ 卡片1   │      │ 卡片2   │       │
│  │ 宽6.5   │      │ 宽6.5   │       │
│  │ 高3.5   │      │ 高3.5   │       │
│  └─────────┘      └─────────┘       │
│  ┌─────────┐      ┌─────────┐       │
│  │ 卡片3   │      │ 卡片4   │       │
│  │ 宽6.5   │      │ 宽6.5   │       │
│  │ 高3.5   │      │ 高3.5   │       │
│  └─────────┘      └─────────┘       │
└─────────────────────────────────────┘
```

**坐标计算**：
- 上排：中心Y=+2，高度3.5，占据+0.25到+3.75
- 下排：中心Y=-2，高度3.5，占据-3.75到-0.25
- 垂直间距：0.5单位

#### 模式C：列表布局（3卡片）
**适用场景**：步骤展示、层次结构
```
┌─────────────────────────────────────┐
│  ┌───────────────────────────────┐  │
│  │         顶部卡片              │  │

│  └───────────────────────────────┘  │
│  ┌───────────────────────────────┐  │

│  │         高3.0                 │  │
│  └───────────────────────────────┘  │
│  ┌───────────────────────────────┐  │

│  │         高2.0                 │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

#### 模式D：单卡片全屏
**适用场景**：核心概念展示、强调
```

│                                     │
│                                     │

│         宽13.6                      │

│                                     │

```

### 第3步：验证不重叠

**检查清单**：
- [ ] 水平方向：卡片A右边界 + 间距 ≤ 卡片B左边界
- [ ] 垂直方向：卡片A下边界 + 间距 ≤ 卡片B上边界
- [ ] 间距至少0.2单位（推荐0.4-0.5）
- [ ] 所有卡片在14×8屏幕内

**Python验证代码**：
```python
def check_no_overlap(card1, card2, min_gap=0.2):
    """检查两个卡片是否重叠"""
    # 水平方向
    c1_right = card1.get_center()[0] + card1.width/2
    c1_left = card1.get_center()[0] - card1.width/2
    c2_right = card2.get_center()[0] + card2.width/2
    c2_left = card2.get_center()[0] - card2.width/2
    
    # 垂直方向
    c1_top = card1.get_center()[1] + card1.height/2
    c1_bottom = card1.get_center()[1] - card1.height/2
    c2_top = card2.get_center()[1] + card2.height/2
    c2_bottom = card2.get_center()[1] - card2.height/2
    
    # 检查是否重叠
    horizontal_overlap = not (c1_right + min_gap <= c2_left or c2_right + min_gap <= c1_left)
    vertical_overlap = not (c1_bottom - min_gap >= c2_top or c2_bottom - min_gap >= c1_top)
    
    if horizontal_overlap and vertical_overlap:
        return False, "卡片重叠！"
    return True, "间距安全"
```

### 第4步：卡片内布局

**原则**：
- 文字可以小（24-36pt）
- 图形可以大（占满卡片80-90%）
- 元素不超出卡片边界
- 使用相对定位（基于卡片中心/边缘）

**常用布局**：
```python
# 居中放置
element.move_to(card.get_center())

# 顶部对齐
element.move_to(card.get_top() + DOWN * 0.5)

# 底部对齐
element.move_to(card.get_bottom() + UP * 0.5)

# 左侧对齐
element.move_to(card.get_left() + RIGHT * 0.5)

# 右侧对齐
element.move_to(card.get_right() + LEFT * 0.5)
```

## 布局选择指南

| 场景类型 | 推荐布局 | 说明 |
|---------|---------|------|
| 概念引入 | 单卡片全屏 | 突出核心概念 |
| 可视化+说明 | 左右分屏 | 左图右文，经典布局 |
| 对比展示 | 网格布局 | 并排对比多个对象 |
| 步骤讲解 | 列表布局 | 按步骤从上到下 |
| 多角度分析 | 网格布局 | 2×2展示不同视角 |
| 复杂推导 | 左右分屏或列表 | 公式和图形配合 |

## 调试技巧

1. **显示边框**：`card.set_stroke(RED, width=2, opacity=0.3)`
2. **检查边界**：在卡片角落放置小点标记
3. **动画预览**：先用`self.wait()`冻结查看布局
4. **坐标打印**：`print(f"Card center: {card.get_center()}")`
