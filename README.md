# OpenCode Skills 仓库

AI Agent 技能集合仓库，包含多种专用能力模块，可与 OpenCode 等 AI 编程助手配合使用。

## 技能列表

| 技能 | 说明 |
|------|------|
| **smart-search** | 整合 DDGS、Tavily、Brave Search 三大引擎的智能搜索 |
| **aw-diary** | 基于 ActivityWatch 数据的每日活动轨迹日记与生产力分析 |
| **pdf-skill** | PDF 全面处理：读取、提取、转换、合并、拆分、OCR 等 |
| **md2pdf** | Markdown 转 PDF，支持 5 种精美模板和中文排版 |
| **memos-skill** | Memos 笔记系统完整管理（增删改查、标签、附件、评论） |
| **obsidian-cli-wsl** | 通过 CLI 与本地 Obsidian 交互（WSL 环境） |
| **sixdim-prompt** | 六维框架专业提示词生成器 |

## 使用方法

克隆到 OpenCode 的 skills 目录：

```bash
git clone git@github.com:WuYuStar/opencode-skills.git ~/.agents/skills
```

每个技能目录包含 `SKILL.md` 文件，详细说明用途、触发条件和具体用法。
